# Meu Primeiro Repositório

Repositório criado para a atividade prática da disciplina de **Engenharia de Software III**, com o objetivo de explorar as funcionalidades da plataforma GitHub como ferramenta de controle de versão e hospedagem de projetos.

## 📌 Sobre o projeto

Este projeto consiste em uma página web simples desenvolvida em HTML, utilizada como base para experimentar o fluxo de trabalho do GitHub: criação de repositório, commits, branches, pull requests e merges. A escolha de um projeto simples permite focar no aprendizado das funcionalidades da plataforma, sem que a complexidade do código desvie a atenção do objetivo principal.

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura da página
- **Git** — sistema distribuído de controle de versão
- **GitHub** — hospedagem do repositório e plataforma de colaboração

## 📁 Estrutura do repositório

```
primeiro-repositorio/
├── index.html      # Página principal
└── README.md       # Documentação do projeto
```

## 🚀 Como visualizar

Para visualizar a página localmente:

1. Faça o download do repositório clicando em **Code → Download ZIP**
2. Extraia o arquivo `.zip` em uma pasta de sua preferência
3. Abra o arquivo `index.html` em qualquer navegador

## 🌿 Branches

- **main** — versão estável do projeto
- **desenvolvimento** — branch criada para alterações experimentais, integrada à `main` por meio de pull request

## 📚 Funcionalidades do GitHub exploradas

- [x] Criação de repositório público
- [x] Inicialização com README.md
- [x] Criação de arquivos pela interface web
- [x] Commits com mensagens descritivas
- [x] Criação de branches
- [x] Pull requests
- [x] Merge de branches
